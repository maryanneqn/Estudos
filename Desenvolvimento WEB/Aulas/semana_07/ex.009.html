No livro PHP: a Bíblia, Tim, Converse explica o que é PHP.

O PHP é uma linguagem para a criação de scripts para a Web do lado servidor embutidos em HTML,
cujo código-fonte é aberto, e que pe compatível com os mais importantes servidores Web
(especialmente Apache). O PHP permite incorporar fragmentos de código em página HTML
normais-código esse que é interpretado à medida que suas páginas são oferecidas aos usuários. O
PHP também serve como uma linguagem que "cola", facitando a conexão de suas páginas Web com
o banco de dados do lado servidor.
